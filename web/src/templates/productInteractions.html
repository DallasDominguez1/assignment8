Key Product Interactions

  • Power On/Off: Toggling Power for the Scale even when plugged into a wall outlet.
  • Unit Change: A button to change between grams and ounces for measurement.
  • Tare: A button to zero the weight on the scale.
  • A Window to display nutritional information from a data table.
  • A button to add the metrics from the current item to the previous one.
  • A button to remove an item.
  • A button to show the sum.
  • A window for user input to choose the correct item if the model is uncertain.
